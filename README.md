# From-Bangumi-to-Obsidian
快速从Bangumi导入条目信息至Obsidian,进而搭建本地动漫库

## 前置需求:

本地Python环境

## 使用方法:

1. 前往Release界面,下载最新.zip文件,解压缩,进入根目录打开cmd
2. 输入`python Start.py`
3. 依照提示输入条目id即可

## 获取条目ID:

进入Bangumi对应条目界面,上方地址栏`https://bangumi.tv/subject/*或https://bgm.tv/subject/*`*即为条目ID
